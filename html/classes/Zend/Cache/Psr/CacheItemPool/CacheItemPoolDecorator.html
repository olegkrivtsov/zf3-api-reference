Class CacheItemPoolDecorator

Summary

Fully Qualified Name: Zend\Cache\Psr\CacheItemPool\CacheItemPoolDecorator
Implements: CacheItemPoolInterface

Description

Decorate zend-cache adapters as PSR-6 cache item pools.
